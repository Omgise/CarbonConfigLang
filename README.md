# Carbon Config Language Files

This is the repository for all Language files for Carbon Config.

### Legacy Entries

The language files contain "Legacy" entries.    
These are mainly for translations that Minecraft started to provide in later versions but not in earlier versions.

So we have to port them back.     
But to ensure we don't replace existing translations, if Minecraft ever changes things, we give them the legacy prefix to only apply them in the older Minecraft versions.
